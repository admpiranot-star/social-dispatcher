🎯 PRÓXIMOS PASSOS IMEDIATOS
═══════════════════════════════════════════════════════════════════════════════

1. ✅ [JÁ FEITO] npm install
2. ⏭️  [PRÓXIMO] Configurar Canva:
   - Ir em https://www.canva.com/developers/
   - Gerar API Key
   - Adicionar ao .env.social:
     export CANVA_API_KEY="pk-..."

3. ⏭️  Criar templates no Canva (por categoria)
4. ⏭️  Rodar Social Dispatcher:
   npm run dev (Terminal 1)
   npm run worker:dev (Terminal 2)

5. ⏭️  Testar POST sem imagem:
   curl -X POST http://localhost:3302/api/dispatch ...
   (ver CANVA-SETUP.md para exemplos)
